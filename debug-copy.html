Copy Button Debug Information

Quick Checks:

  1. Is the canvas initialized? - The copy button only appears after you upload an image or start drawing
  2. Are you using HTTPS or localhost? - The Clipboard API requires a secure context
  3. Browser support: - Chrome, Edge, Safari (recent versions) support image copying. Firefox has limited support.

The Copy Button Location:

The copy button is the circular button with the copy icon in the top-right corner of the header, next to the user profile menu.

How it works:

  • If shapes are selected: It copies the selected shapes (for pasting within the app)
  • If no shapes are selected: It copies the entire canvas as an image to your system clipboard

Testing the Copy Feature:

  1. Upload an image or draw something on the canvas
  2. Click the copy button in the top-right corner (circular button with copy icon)
  3. You should see the button turn green and show "Copied!" for 2 seconds
  4. Try pasting in another application (e.g., MS Paint, Photoshop, or any image editor)

Alternative Copy Methods:

  • Keyboard shortcut: Ctrl+C (or Cmd+C on Mac) when canvas has focus
  • Edit Menu: Click Edit > Copy in the menu bar

If it's not working:

  • Open the browser console (F12) and look for error messages
  • Make sure you're on localhost or HTTPS
  • Try a different browser (Chrome/Edge recommended)
  • Check if your browser has clipboard permissions enabled
